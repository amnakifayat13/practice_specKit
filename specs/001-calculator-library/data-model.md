# Data Model for Calculator Library

The data model for this feature is simple and does not involve complex entities or storage.

## Entities

-   **Numeric Input**: The primary data inputs are numeric types, specifically Python's `int` and `float`. The library functions will operate on these types.

## Validation Rules

-   Inputs must be instances of `int` or `float`. A `TypeError` will be raised for any other type.
